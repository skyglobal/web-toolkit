Video

A wrapper for the Sky video player, in this example it's embedded onto an image. The video will expand the parent container to fit the 16/9 ratio of the video.
Ensure that you include all the needed dependencies (htmlplayer.js and player css). Check the dependencies tab.
Example: $(document).on('click','.play-video', function(e) { e.preventDefault(); var videoOptionsObject = { token:"Your 36 long SkyVideo Token", autoplay: true, videoId: $(this).attr('data-video-id'), freewheel: false, placeHolderImage: '' }, video, videoContainer = $(this).parents('.video'); video = new toolkit.video(videoContainer, videoOptionsObject); video.play(); });
