With accessible error messages for required fields:

This requires the use of the native required attribute on an input field. See below for an example.

<input id="my-input" type="text" required/>

If you are following xhtml, use the following:

<input id="my-input" type="text" required="required"/>

Alternatively, if you want to customise the error label, you can provide a custom error label as in the example below.

<input id="my-input" type="text" required/><label class="hidden form-error feedback" for="input">Please enter some text</label>

Browsers that support the required attribute will get their native experience, so this logic can only be observed on browsers that do not support it (IE8, IE9, Safari)
